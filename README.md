# 📚 PeerConnect: Study & Book Exchange Matchmaker

## 🚀 Overview

**PeerConnect** is a Java-based application designed to help students **find compatible study partners** and **exchange books locally**. The platform focuses on improving collaboration and resource sharing among students by matching users based on their subjects, preferences, and goals.

---

## 🎯 Features

* 👤 **User Profiles** – Create and manage personalized study profiles
* 🤝 **Peer Matching System** – Find ideal study partners based on interests
* 📚 **Book Exchange Module** – List, request, and exchange books
* 📊 **Leaderboard & Achievements** – Track engagement and progress
* ✅ **Task & Checklist System** – Manage study goals efficiently
* 🎨 **Modern UI (Java Swing)** – Clean and interactive interface

---

## 🛠️ Tech Stack

* **Language:** Java
* **UI:** Java Swing
* **Database:** SQLite
* **Libraries:**

  * FlatLaf (UI styling)
  * SQLite JDBC Driver
  * SLF4J (Logging)
